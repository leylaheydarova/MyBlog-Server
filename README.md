Project Overview

This is a Mini Blog project designed for beginner ASP.NET MVC students to practice integrating frontend and backend development. The project demonstrates core concepts such as:

MVC architecture (Model-View-Controller)

User authentication and authorization

Blog CRUD operations (Create, Read, Update, Delete)

Favorites management

Personal profile page

The frontend is built using HTML, CSS, and JavaScript and connects to a backend implemented in ASP.NET Core MVC.
